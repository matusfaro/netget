Speak network. Skip the boilerplate._

Spin up network servers and clients using plain English. 50+ protocols. LLM figures out the rest.

netget
> listen on 8080 via HTTP, serve sassy responses
[INFO] HTTP server listening on 0.0.0.0:8080
[INFO] GET /api/status from 127.0.0.1
[LLM] "Oh, you want status? Server's up. You're welcome."

Three Response Modes

Pick your tradeoff: intelligence vs speed vs predictability

LLM Mode

Dynamic, context-aware responses. Can read files and search the web.

> DNS server on 5353
> look up RFC 1035 and respond correctly
> resolve *.local to 127.0.0.1

Static Mode

Fixed responses. Zero latency, zero LLM calls after setup.

> HTTP server on 3000
> /health returns {"status": "ok"}
> /version returns {"v": "1.2.3"}

Scripting Mode

Python/JavaScript handlers. Full control, deterministic.

> HTTP server on 4000, use scripting
> /time returns: new Date().toISOString()
> /random returns: Math.random()

Examples

SSH Honeypot

> SSH server on 2222, pretend to be Ubuntu 22.04
> accept any password, log everything
> when they run "cat /etc/passwd", show fake users

Mock Database

> MySQL server, read schema.json for structure
> answer SELECT queries with realistic fake data
> reject DROP and DELETE with "nice try"

Protocol from RFC

> TCP server on 4545
> search for RFC 959 (FTP) and implement it
> serve files from ./public directory

API with Personality

> HTTP on 8080, you're a grumpy sysadmin
> answer questions about Linux but complain about it
> if asked about Windows, refuse dramatically

Capabilities

Web Search - LLM can look up RFCs, docs, or any info to respond correctly
File Access - Read local files to serve content or use as context
Memory - Remembers conversation context across requests
Scheduled Tasks - Cron-like actions: broadcast, cleanup, keepalives
Both Directions - Run as server OR client for any protocol

Get Started

# Install Ollama
curl -fsSL https://ollama.ai/install.sh | sh
ollama pull qwen3-coder:30b

# Build NetGet
git clone https://github.com/matusfaro/netget.git
cd netget && cargo build --release --features tcp,http,ssh,dns

# Run
./target/release/netget

Requirements: Rust, ~8GB RAM for LLM
Tip: Use --features X,Y,Z for faster builds
