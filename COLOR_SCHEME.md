# NetGet Color Scheme

## Overview

NetGet uses a **Midnight Commander inspired blue theme** - the classic blue background with cyan/white text that brings back memories of DOS-era file managers!

## Panel Color Breakdown

### 1. User Input Panel (Bottom - 30%)

```
┌─ User Input (Press Enter to submit, Ctrl+C to quit) ────┐ ← BOLD CYAN border
│ listen on port 21 via ftp                               │ ← WHITE text on BLUE bg
└──────────────────────────────────────────────────────────┘
```

- **Border**: Bold Cyan (`Color::Cyan` + `Modifier::BOLD`)
- **Text**: White on Blue background (`fg: White, bg: Blue`)
- **Background**: Blue (`bg: Blue`)
- **Purpose**: Classic Midnight Commander look

## Terminal Compatibility

### ✅ Recommended Terminals

- **iTerm2** (macOS) - Excellent support
- **Alacritty** - Full color support
- **kitty** - Excellent rendering
- **Windows Terminal** - Good support
- **GNOME Terminal** (Linux) - Good support
- **Konsole** (KDE) - Good support

### ⚠️ Limited Support

- **Terminal.app** (macOS default) - Works but limited colors
- **PuTTY** - May have color issues
- **CMD.exe** (Windows) - Limited color palette

### 💡 Best Experience

For the best visual experience:
1. Use a terminal with **24-bit true color** support
2. Set your terminal background to **dark/black**
3. Use a modern monospaced font (e.g., Fira Code, JetBrains Mono)
4. Enable terminal **bold font rendering**

## Accessibility Considerations

### High Contrast

All text has been chosen to provide **high contrast** against black backgrounds:

- **Cyan** on black: 7:1 contrast ratio
- **Green** on black: 6:1 contrast ratio
- **Yellow** on black: 6:1 contrast ratio
- **Magenta** on black: 5:1 contrast ratio
- **Black** on white: 21:1 contrast ratio (input panel)

### Color Blindness

Colors chosen to be distinguishable for common color vision deficiencies:

- **Deuteranopia** (red-green): Cyan/Magenta/Yellow remain distinct
- **Protanopia** (red-green): Similar to above
- **Tritanopia** (blue-yellow): Green/Cyan/Magenta remain distinct

## Customization

To modify colors, edit `src/ui/app.rs`:

```rust
// Example: Change LLM text from Cyan to Yellow
.style(Style::default().fg(Color::Yellow))

// Example: Change border from Green to Blue
.border_style(Style::default().fg(Color::Blue).add_modifier(Modifier::BOLD))
```

Available colors:
- `Color::Black`, `Color::Red`, `Color::Green`, `Color::Yellow`
- `Color::Blue`, `Color::Magenta`, `Color::Cyan`, `Color::White`
- `Color::LightRed`, `Color::LightGreen`, `Color::LightYellow`
- `Color::LightBlue`, `Color::LightMagenta`, `Color::LightCyan`
- `Color::Gray`, `Color::DarkGray`
- `Color::Rgb(r, g, b)` for custom colors
