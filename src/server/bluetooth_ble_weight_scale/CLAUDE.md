# BLE Weight Scale Service

Standard Weight Scale Service (0x181D) for health/fitness tracking.

## Actions
- `set_weight` - Set weight measurement (kg or lbs)
- `set_bmi` - Set Body Mass Index
- `multi_user` - Support multiple user profiles

## Service UUID: 0x181D
- Characteristic 0x2A9D: Weight Measurement (indicate)
