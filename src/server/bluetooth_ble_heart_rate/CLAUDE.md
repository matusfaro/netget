# BLE Heart Rate Service

Standard Heart Rate Service (0x180D) - reports beats per minute (BPM).

## Actions
- `set_bpm` - Set heart rate (30-220 BPM)
- `simulate_activity` - Simulate exercise (rest/walk/jog/run/sprint)
